# 🌍 WanderLust - Airbnb Clone (EJS Version)

WanderLust is a full-stack Airbnb Clone project built using:

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Cloudinary

Users can:

✅ Create Listings  
✅ Upload Images  
✅ Add Reviews  
✅ Login & Signup  
✅ Search Destinations  
✅ Edit/Delete Listings  

---
